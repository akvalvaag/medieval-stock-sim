System Architecture

Spring Boot backend + Vue.js frontend, communicating via WebSocket and REST

Backend — Spring Boot
  • MarketEngine — ticks every 5s, updates prices
  • PriceModel — drift + supply/demand + events
  • EventEngine — randomly fires market events
  • WebSocket broker — broadcasts price ticks to clients
  • REST API — buy/sell actions, initial state
⇄
Frontend — Vue.js
  • ClassSelect view — pick Merchant / Miner / Noble
  • Market view — live price table, buy/sell controls
  • Portfolio panel — holdings, gold, net worth
  • Event ticker — scrolling news of market events
  • localStorage — saves game state on every action
Data Flow
MarketEngine tick (every 5s)
  → PriceModel computes new prices
  → WebSocket broadcasts /topic/market
  → Vue receives update, re-renders price table

Player clicks Buy/Sell
  → REST POST /api/trade
  → Server updates supply/demand, returns new state
  → Vue updates portfolio + localStorage
